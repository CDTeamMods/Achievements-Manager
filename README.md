### 📋 **Pré-requisitos**

- Node.js 22+ 
- Yarn 1.22.22+

### 🔧 **Instalação**

```bash
# Clone o repositório
git clone https://github.com/CDTeamMods/Achievements-Manager.git

# Entre no diretório
cd Achievements-Manager

# Instale as dependências
yarn

# Execute em modo desenvolvimento
yarn dev
```

### 📦 **Build para Produção**

```bash
# Build geral
yarn build

# Build específico por plataforma
yarn build:win    # Windows
```
